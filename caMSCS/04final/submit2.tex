\opening{ Dear Prof. Giuseppe Longo,}
Please, find enclosed a new copy of
the paper {\em ``Computation Algebras''} which has now been formatted using
the {\sf mscs.cls} style in \LaTeX $2_\varepsilon$.

The main problem which I encountered using this style concerns footnotes --
at some point the error `Counter too large' is being reported and the
subsequent footnotes (starting from page 27) are not marked. I do not
understand the idea of the strange marks used for footnotes and I hope that
the editing staff manages to figure out (from the source code or the original hard
copy) the appropriate places
in the text where these footnotes should be attached.

Following the guidelines, all the formatting commands ({\sf vspace}, etc.),
placing of figures ({\sf [hbt]}) and the like
have been removed. I am far from satisfied with the result and in case of
doubt I would appreciate if one checked the original hard copy rather than
the one enclosed here. 

The enclosed PC disc contains the \LaTeX $2_\varepsilon$-source files:
\begin{itemize}
\item the main file {\sf ca04final.tex} and
\item the file {\sf defs.tex} with the macros used.
\end{itemize}
